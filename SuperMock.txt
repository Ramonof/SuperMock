Слайд 1
LaMock Cloud - это платформа для быстрого создания высококачественных симуляций производственных сервисов
с сохранением состояния, которые работают как простые в использовании имитационные API.

Вы можете использовать эти моделируемые службы локально(local), в средах разработки(dev) и в тестовых средах(test),
чтобы заменить зависимости API производства, которые замедляют работу и стоят денег.

1. Совместная быстрая разработка прототипов нового API
2. Запись и ипморт API зависимостей
3. Точное обнаружения отклонений и хаос тестирование
4. Паралельная разработка

Получи
- Тысячи часов в скорости
- Более быстрый запуск разработки
- Более надежная скорость параллельной разработки

Сократи
- Нестабильность из-за ненадежных API
- Время, проведенное с заблокированными разработчиками
- Упущеные баги + инциденты на проде
- Дорогостоящие затраты на тестовые окружения

Точный (Картинка Stub request/response)
Моделируемые API, отражающие реальность
- Соответствие запросов, ошибки, задержка ответа, динамические ответы
- Динамические состояния во время сеансов тестирования
- Динамические состояния сеансов тестирования и мощные сценарии

Гибкий (Картинка Генератор OpenAPI)
Получите максимум от OpenAPI
- Храните файлы OpenAPI в Git, отправляйте и извлекайте изменения в свои макеты по мере необходимости
- Создание прототипов с использованием макетов для быстрого проектирования новых API
- Автоматическая генерация спецификаций OpenAPI

Надежный (Картинка Хаос)
Более отказоустойчивое программное обеспечение
- Chaos Engineering проактивно обнаруживает проблемы до того, как они станут причиной проблем.
- Простое, масштабное нагрузочное тестирование с точными ограничениями скорости, установленными сторонними сервисами.
- Автоматическое обновление фиктивных тестов при изменении API в реальном времени для предотвращения ложных срабатываний безопасности.

Совместный (Картинка Пользователи и группы в проекте)
Работайте в команде
-Обмен шаблонами и макетами внутри компании
-Быстрое создание прототипов новых API → макеты и возможность параллельной разработки
-Мгновенное предоставление документации через портал разработчиков

Расширяемый (Картинка Темплейты популярных сервисов aws, stripe, github, etc)
Создано для серьёзного бизнеса
-SOC 2, RBAC, OpenID Connect и журнал запросов
-Готовые шаблоны для тысяч критически важных сторонних API
-Размещение в облаке или самостоятельное управление через Kubernetes

Фичи
- Static Mocking
- API Recording
- API Templates
- Advanced Dynamic Responses
- Chaos Testing
- External Data Sources
- Stateful Mocking


Слайд 2
Обзор

Платформа LaMock Cloud предоставляет ряд функций корпоративного уровня, о которых вы можете узнать больше в документации здесь.
- Простая разработка, импорт или запись любого нового виртуализированного API
- Управление командой с помощью SSO и RBAC
- Журнал активности для мониторинга использования сервисов
- Использование Git в качестве источника достоверной информации для спецификаций API
- Внедрение ошибок, хаоса и других форм поведения «на черный день»
- Создание синтетических тестовых данных или импорт данных из CSV-файла или базы данных

Создани API

LaMock Cloud предоставляет несколько способов настройки фиктивного API:
- Вручную через веб-приложение
- Импорт OpenAPI, Swagger, Postman или HAR
- Запись трафика к другому API и из него
- Импорт существующего проекта из WireMock OSS
- Автоматизация с помощью собственных REST API LaMock
